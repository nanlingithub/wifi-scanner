## 📥 部署步骤

### 方式一：直接运行（推荐）
1. 将整个 `WiFi专业工具` 文件夹复制到目标计算机
2. 右键点击 `WiFi专业工具.exe`
3. 选择"以管理员身份运行"

### 方式二：创建桌面快捷方式
1. 右键拖动 `WiFi专业工具.exe` 到桌面
2. 选择"在当前位置创建快捷方式"
3. 右键快捷方式 → 属性
4. 勾选"以管理员身份运行此程序"
5. 点击确定

### 方式三：制作安装包（可选）
使用 Inno Setup 或 NSIS 等工具制作安装程序：
- 自动添加桌面快捷方式
- 自动设置管理员权限
- 添加到开始菜单
- 提供卸载功能

## ⚡ 首次运行

### 正常启动流程
1. 双击程序后会看到 Windows 安全提示
2. 点击"更多信息" → "仍要运行"
3. UAC提示时选择"是"
4. 程序启动可能需要3-5秒（首次较慢）
5. 主界面打开后即可正常使用

### 可能遇到的问题

#### 1. 杀毒软件拦截
**现象**: 360、Windows Defender等报告威胁

**解决方案**:
- 添加程序到信任列表
- 或临时关闭实时保护
- 文件是安全的，误报是因为PyInstaller打包特征

#### 2. 缺少管理员权限
**现象**: 无法扫描WiFi网络

**解决方案**:
- 右键 → 以管理员身份运行
- 或在快捷方式属性中设置

#### 3. 程序无法启动
**现象**: 双击没有反应

**解决方案**:
- 检查 `_internal` 文件夹是否完整
- 重新解压或复制完整文件夹
- 检查系统是否为64位

## 🔧 配置文件

### config.json
位置: `dist/WiFi专业工具/config.json`

```json
{
  "scan_interval": 5,
  "theme": "light",
  "language": "zh-CN",
  "save_history": true
}
```

可修改参数：
- `scan_interval`: 扫描间隔（秒）
- `theme`: 主题（light/dark/enterprise_blue等）
- `language`: 语言
- `save_history`: 是否保存历史记录

## 🆘 故障排除

### 问题1: 提示找不到python311.dll
**解决**: 确保 `_internal` 文件夹完整，重新复制整个文件夹

### 问题2: 程序闪退
**解决**: 
1. 以管理员身份运行
2. 检查config.json是否损坏
3. 删除config.json让程序自动创建

### 问题3: 无法扫描WiFi
**解决**:
1. 确认有WiFi网卡
2. 以管理员身份运行
3. 检查WiFi是否已开启

### 问题4: 热力图无法生成
**解决**:
1. 至少需要3个以上的信号点
2. 确保有足够内存
3. 稍后重试
