## 🚀 快速开始

### 方式一：使用启动脚本（推荐）
1. 双击 `启动WiFi专业工具.bat`
2. 在UAC提示中选择"是"
3. 等待程序启动（首次3-5秒）

### 方式二：直接运行
1. 进入 `dist\WiFi专业工具\` 文件夹
2. 右键 `WiFi专业工具.exe`
3. 选择"以管理员身份运行"

### 方式三：创建快捷方式
1. 右键拖动 `WiFi专业工具.exe` 到桌面
2. 右键快捷方式 → 属性
3. 勾选"以管理员身份运行此程序"

## ⚠️ 常见问题

### Q1: 程序无法启动？
**A**: 请确保：
1. 以管理员身份运行
2. `_internal` 文件夹完整
3. 系统为64位Windows

### Q2: 杀毒软件报警？
**A**: PyInstaller打包的程序可能被误报，请添加到信任列表。程序完全安全，源代码可审查。

### Q3: 扫描不到WiFi？
**A**: 请确保：
1. 计算机有WiFi网卡
2. WiFi功能已开启
3. 以管理员身份运行

### Q4: 无法生成报告？
**A**: 检查：
1. 是否有足够磁盘空间
2. 报告保存路径是否有写入权限
