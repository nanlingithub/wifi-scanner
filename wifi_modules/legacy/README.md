# Legacy版本说明

## realtime_monitor_v1.0_deprecated.py

**状态**: ⚠️ 已废弃 (Deprecated)

**原因**: 
- 性能瓶颈: 频谱渲染220ms（优化版35ms，快84%）
- 内存限制: 1000条数据上限，仅支持16分钟监控
- 缺少高级功能: 无AI预测、异常检测、智能降采样

**替代版本**: `realtime_monitor_optimized.py`

**迁移时间**: 2025年1月

**保留原因**: 代码参考和回滚备份
