## 🔧 使用方法

### 方法1: 菜单切换（推荐）

1. 打开程序
2. 点击菜单栏 **工具 → 🎨 主题选择**
3. 选择您需要的主题
4. 程序立即应用新主题

### 方法2: 快捷键切换

- 暂无快捷键（可根据需求添加）

### 方法3: 配置文件设置

在 `config.json` 中添加：
```json
{
  "theme": "enterprise_blue"
}
```

## 🔧 高级配置

### 自定义主题颜色

如需自定义主题，编辑 `wifi_modules/theme.py`：

```python
# 添加自定义主题
CUSTOM_THEME = {
    'bg': '#ffffff',           # 背景色
    'fg': '#000000',           # 文字色
    'primary': '#0066cc',      # 主色调
    'card_bg': '#ffffff',      # 卡片背景
    # ... 其他配置
}
```

### 主题持久化

程序会记住您上次选择的主题，下次启动自动应用。

## 🆘 常见问题

### Q1: 切换主题后部分界面未更新？
**A**: 部分标签页需要重新打开（关闭后重新选择）才能完全应用新主题。

### Q2: 如何将主题设为默认？
**A**: 修改 `config.json`，添加 `"theme": "enterprise_blue"`。

### Q3: 主题颜色能否自定义？
**A**: 可以，编辑 `theme.py` 文件中的对应主题配置。

### Q4: 主题切换会影响性能吗？
**A**: 不会，主题仅影响UI渲染，不影响WiFi扫描和分析性能。
